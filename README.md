# Smart Irrigation System using IoT

## Overview

The Smart Irrigation System is an IoT-based solution that automates irrigation by monitoring soil moisture levels and controlling a water pump accordingly.

## Features

- Automatic irrigation
- Soil moisture monitoring
- ESP32 based control
- Water conservation

## Components Used

- ESP32
- Soil Moisture Sensor
- Relay Module
- Water Pump

## Working Principle

The soil moisture sensor measures the moisture level of the soil. When the moisture falls below a threshold, the ESP32 activates the relay and turns on the water pump.
